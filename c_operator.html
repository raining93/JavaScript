연산자

        산술 연산자 : + - * / %
        단항 연산자 : ++ --
        비교 연산자 : < > <= >= == != === !==
        논리 연산자 : && ||
        삼항 연산자 : 조건식 ? 값1 : 값2
        문자열 결합 연산자 : +
        승의 문자 ** ex(2**3하면 8이 나옴  9)

동등연산자와 일치 연산자 

        == : type을 추론하여 casting한 다음 casting된 값으로 비교연산을 수행(타입이 달라도 안의 값이 같으면 true를 반환)
        === : type과 값이 정확하게 일치해야만 true

원하는 숫자를 입력하세요.
